# excel-reconciliation-tool
Excel/VBA tool for automated reconciliation of operational reports

## Контекст проекта

Проект был сделан в рамках прохождения практики. В процессе работы мне обозначили проблему: сверка заявок и отчёта по невывозу выполнялась вручную, потому что данные находились в двух отдельных Excel-выгрузках, а один и тот же ID площадки мог встречаться в нескольких строках.

Изначально это не было отдельным формальным заданием. Задача возникла как практический запрос: если получится упростить или автоматизировать этот процесс, это поможет в ежедневной работе.

В результате был собран Excel/VBA-инструмент, который позволяет пользователю выбрать два входных файла и получить готовый итоговый файл со статусами и сводкой.

В репозитории используются только демонстрационные данные. Реальные рабочие выгрузки, адреса, идентификаторы и внутренняя информация не публикуются.

## Задача

В исходном процессе сотрудники ежедневно работают с двумя Excel-файлами:

1. Файл заявок — содержит плановый объём вывоза по площадкам по Московской области.
2. Отчёт по невывозу — содержит объёмы, которые не были вывезены полностью или частично.

Один и тот же ID площадки может встречаться в нескольких строках, поэтому ручная сверка требует группировки данных, расчёта сумм и проверки статусов.

## Решение

Разработан Excel-файл с кнопкой запуска макроса. Пользователь выбирает два входных файла, после чего инструмент автоматически:

- находит нужные столбцы по заголовкам;
- группирует заявки по `ID площадки`;
- группирует отчёт по невывозу по `ID МНО`;
- считает плановый объём по каждой площадке;
- считает объём невывоза;
- рассчитывает фактически вывезенный объём;
- присваивает статус: `вывоз`, `частичный вывоз`, `невывоз`;
- формирует итоговый лист `Итог по площадкам`;
- строит сводку по видам отходов и статусам.

## Логика статусов

| Условие | Статус |
|---|---|
| ID площадки отсутствует в отчёте по невывозу | вывоз |
| Объём невывоза равен плановому объёму | невывоз |
| Объём невывоза меньше планового объёма | частичный вывоз |
| Объём невывоза больше планового объёма | невывоз + комментарий для проверки |

## Используемые технологии

- Microsoft Excel
- VBA
- Обработка Excel-выгрузок
- Группировка и агрегация данных
- Автоматизация операционной отчётности

## Результат

Инструмент сокращает ручную сверку до выбора двух файлов и нажатия одной кнопки.

На выходе формируется Excel-файл с итоговым статусом по каждой площадке и сводкой по видам отходов.
