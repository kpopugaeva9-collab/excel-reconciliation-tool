# User Manual

## Как пользоваться инструментом

1. Скачайте файл с заявками.
2. Скачайте файл с отчётом по невывозу.
3. Закройте оба скачанных файла, если они открыты.
4. Откройте файл с кнопкой.
5. Нажмите кнопку “СФОРМИРОВАТЬ ОТЧЁТ”.
6. В первом окне выберите файл с заявками.
7. Во втором окне выберите файл с отчётом по невывозу.
8. Дождитесь сообщения “Готово”.
9. Откройте созданный итоговый файл.
10. Перейдите на лист “Итог по площадкам”.

## Что делает инструмент

Инструмент сверяет две Excel-выгрузки:

- файл заявок;
- отчёт по невывозу.

После обработки создаётся итоговый файл, в котором по каждой площадке рассчитываются:

- плановый объём;
- объём невывоза;
- фактически вывезенный объём;
- статус факт: `вывоз`, `частичный вывоз`, `невывоз`.

Также формируется сводка по видам отходов и статусам.

## Важно

Перед запуском входные файлы должны быть закрыты.

Если Excel просит разрешить макросы, их нужно разрешить, иначе кнопка не сработает.

В репозитории используются только демонстрационные данные.
